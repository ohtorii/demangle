C/C++のシンボル名をデマングルする秀丸マクロ
========

###デマングル(demangle)できる形式
- GCC v3
- VisualStudio


###動作環境
- 秀丸エディタ ver8以降
- 田楽(でんがく)ＤＬＬ [http://www.ceres.dti.ne.jp/~sugiura/](http://www.ceres.dti.ne.jp/~sugiura/)

###導入方法
下記ファイルを秀丸エディタのマクロディレクトリへコピーしてください、あとは適当にキーアサインしてお使い下さい。

- demangle.exe
- demangle.mac
